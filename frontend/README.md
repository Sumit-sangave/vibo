# frontend (moodMixDJ)

This is a minimal Vite + React frontend for moodMixDJ.

Run locally:

```powershell
cd frontend
npm install
npm run dev
```

Build for production:

```powershell
npm run build
# serve the `dist` folder or deploy to Vercel
```
